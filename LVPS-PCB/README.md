# LVPS-PCB

Project.

The LVPS-PCB support these AC-DC modules:

- Meanwell IRM-15

- Meanwell IRM-30

- Hi-Link HLK-15M

- Hi-Link HLK-20M

- Hi-Link HLK-40M

- Recom RACM30
