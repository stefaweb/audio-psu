# LVPS-PCB

The LVPS-PCB allows the use of various AC-DC modules from Meanwell, Recom or Hi-Link up to 40W.

The LVPS-PCB support these AC-DC modules:

- Meanwell IRM-15

- Meanwell IRM-30

- Hi-Link HLK-15M

- Hi-Link HLK-20M

- Hi-Link HLK-40M

- Recom RACM30
