# HV-MODULE-UC3845-1.0

DC-DC High Voltage 50W Power Supply 

Input: 24V<br>
Positive Output: +180V to +500V<br>
Negative Output: -160V (no load, only for tube BIAS)<br>
<br>

Under development.

Size: 80x50mm
