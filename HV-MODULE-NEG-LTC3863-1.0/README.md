# HV-MODULE-NEG-LTC3863-1.0

<b>High Voltage DC-DC Power Supply for Negative Grid Voltage</b>

Under development.

<b>Specifications:</b><br>
<br>
Input: 12V-24V<br>
Negative Output: -8V to -90V<br>
Noise and ripple: below 1mV RMS<br>
Two independant output<br>
Input for cathode<br>
Output for external probe<br>
Overcurent protection<br>
Overtemperature protection<br>
Softstart<br>
Inverting polarity diode protection on power input<br>

Size: 60 x 50 mm
